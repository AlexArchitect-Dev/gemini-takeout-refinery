A high-efficiency Python utility designed to transform raw Google Gemini Takeout data into a structured, AI-ready format.

## Features
- **Data Reversal:** Corrects the chronological order of chat logs.
- **HTML Sanitization:** Strips unnecessary tags for clean text analysis.
- **Database Ready:** Prepares data blocks with unique identifiers for SQL injection.
- **Portable:** Runs locally to ensure your private data stays private.

## How to Use
1. Clone the repository.
2. Place your `raw.json` in the project folder.
3. Run `python refinery.py`.
